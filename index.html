Автоматизация
▾
Шаги флоу
⚙️
День −5 до оплаты
Дружелюбное напоминание: "Привет! Напоминаем, аренда €1200 — 1-го числа. Реквизиты: ..."
Мессенджер или Email — по предпочтению арендатора

День +3, не оплачено
Нейтральный тон: "Видим, что оплата ещё не поступила. Всё ли в порядке?"
⚠️
День +7 — эскалация к менеджеру
Менеджер получает уведомление: "Квартира 4Б — просрочка 7 дней, €1200"
Официальное напоминание по email с юридическими ссылками отправляется автоматически

Очень низкая

Задача Автоматизация AI-агент (LLM) Почему
Напоминание об оплате ✅ Авто не нужен Чистая логика: дата → условие → действие
Уведомление клинингу при выезде ✅ Авто не нужен Событие → уведомление. Никакого понимания текста
Генерация / деактивация кода замка ✅ Авто не нужен API-вызов по триггеру — детерминированный процесс
Сбор данных для дайджеста ✅ Авто не нужен Агрегация из источников — задача для workflow
Написание дайджеста / отчёта не умеет ✅ LLM Нужно понять данные и написать связный текст
Анализ фото прогресса / клининга не умеет ✅ Vision Понимание изображений — только мультимодальный LLM
Ответ арендатору на вопрос не подходит ✅ LLM Нужно понять намерение и ответить естественно
Извлечь данные из контракта (PDF) не умеет ✅ LLM Неструктурированный текст → структура — задача LLM
Классификация заявки на ремонт только ключевые слова ✅ LLM Понять суть, срочность, тип — семантика, не keyword
Проверка счёта vs контракт только цифры ✅ LLM Нужно сопоставить документы по смыслу, не только суммам
Маршрутизация между системами ✅ Авто избыточен IF/ELSE логика + API-вызовы — чистый workflow
Генерация письма-претензии только шаблон ✅ LLM Контекстно-зависимый текст с нюансами ситуации
💡 Главное правило
Автоматизация — когда:
• Чёткий триггер (время, событие)
• Детерминированный результат
• Передача данных между системами
• Условная логика (если → то)
• Работа с API без анализа текста
AI-агент (LLM) — когда:
• Нужно понять смысл текста
• Неструктурированные данные (PDF, фото)
• Написать уникальный текст
• Принять решение в неопределённости
• Многоязычная коммуникация
